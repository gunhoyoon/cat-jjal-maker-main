# 사용 기술

> React 기반 프로젝트입니다.

fetch + async await 로 데이터 패칭하였습니다.

> 하트를 눌러 좋아하는 고양이를 추가하게 되면 배열에 푸시하는 구조를 만들었습니다.

이 과정에서 불변성을 위해 스프레스 연산자를 사용하였습니다.

## 프로젝트 실행 방법

> cat-jjal-maker-cra 에서 터미널 열어서

> npm install

> npm run start
